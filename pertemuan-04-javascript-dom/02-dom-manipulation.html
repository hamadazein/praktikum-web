DOM Manipulation dengan JavaScript

DOM (Document Object Model) adalah representasi struktur HTML dalam bentuk tree/pohon yang dapat dimanipulasi dengan JavaScript.

1. Memilih Elemen (DOM Selection)

// Memilih elemen berdasarkan ID let element = document.getElementById('myId'); // Memilih elemen berdasarkan class let elements = document.getElementsByClassName('myClass'); // Memilih elemen berdasarkan tag let paragraphs = document.getElementsByTagName('p'); // Modern selectors (lebih fleksibel) let element = document.querySelector('#myId'); // ID let element = document.querySelector('.myClass'); // Class let element = document.querySelector('p'); // Tag let elements = document.querySelectorAll('.myClass'); // Semua elemen dengan class

2. Mengubah Konten Elemen

// innerHTML - mengubah HTML content element.innerHTML = "<strong>Teks tebal</strong>"; // textContent - mengubah text content (tanpa HTML) element.textContent = "Teks biasa"; // innerText - mengubah text yang terlihat element.innerText = "Teks yang terlihat";

Demo: Mengubah Konten

3. Mengubah Atribut

// Mengubah atribut element.setAttribute('class', 'new-class'); element.setAttribute('src', 'image.jpg'); // Mengambil nilai atribut let className = element.getAttribute('class'); // Menghapus atribut element.removeAttribute('class'); // Properti khusus element.id = 'newId'; element.className = 'new-class'; element.src = 'new-image.jpg';

Demo: Mengubah Atribut

4. Mengubah Style/CSS

// Mengubah style inline element.style.color = 'red'; element.style.backgroundColor = 'yellow'; element.style.fontSize = '20px'; element.style.display = 'none'; // Mengubah class element.classList.add('new-class'); element.classList.remove('old-class'); element.classList.toggle('active'); element.classList.contains('existing-class'); // return true/false

// Membuat elemen baru let newElement = document.createElement('div'); newElement.textContent = 'Elemen baru'; newElement.className = 'new-item'; // Menambahkan ke parent parentElement.appendChild(newElement); // Menambahkan sebelum elemen tertentu parentElement.insertBefore(newElement, referenceElement); // Menghapus elemen parentElement.removeChild(childElement); // atau (modern) element.remove();
